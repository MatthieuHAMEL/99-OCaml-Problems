# 99 OCaml Exercises

Link : <https://ocaml.org/exercises>

Those are classical programming exercises in OCaml. They contain classic functional programming problems relating to :

Recursive solutions are given as [tail-recursive](https://en.wikipedia.org/wiki/Tail_call) as much as possible.

- Linked lists : palindromes, flattening, slicing, K-combinations, sorting ...
- Arithmetic : primality test, prime factors, Goldbach conjecture ...
- Logic and Codes : truth tables, Gray code, Huffman code ;
